# 数据源与核验

本文件只说明取数顺序和核验口径，不保存大段代码。

## 取数优先级

1. 交易软件截图中的持仓数据：成本、持股数、持仓盈亏、仓位。
2. 实时行情接口：东方财富、腾讯行情、同花顺等公开行情。
3. 基金/ETF信息：基金公司公告、交易所、天天基金、东方财富基金。
4. 财报与公告：交易所公告、上市公司公告、巨潮资讯。
5. 第三方数据：新闻稿、研报摘要、资金流统计，只作辅助。

## 必须核验

| 字段 | 要求 |
|---|---|
| 日期时间 | 必须与用户问题中的时间窗口一致 |
| 代码 | 必须匹配交易所前缀或市场 |
| 股票名称 | 必须与代码一致，表格不得省略 |
| 现价/涨跌 | 必须来自当日行情或用户截图 |
| 成本/持股数 | 优先使用用户截图 |
| 资金流/申赎 | 必须说明口径，不得混用 |

置信度：
- 已核验：日期、代码、名称、价格一致，可用于决策。
- 部分核验：可辅助判断，不能单独触发操作。
- 未核验：只能提示待确认，不能作为依据。

## ETF资金口径

ETF相关信号要区分：

| 信号 | 含义 | 用法 |
|---|---|---|
| ETF份额/申赎 | 反映一级市场资金进出 | 中期辅助 |
| 成交额/换手率 | 反映二级市场活跃度 | 短线确认 |
| 主力资金 | 统计口径差异较大 | 辅助参考 |
| 溢价率 | ETF价格相对净值偏离 | 判断追高/折价 |

规则：ETF份额增加不等于当天没有抛压；单日主力流出不等于ETF中期资金撤退。

## 常用行情字段

至少获取：

| 字段 | 用途 |
|---|---|
| 最新价、昨收、涨跌幅 | 持仓盈亏和日内强弱 |
| 今开、最高、最低 | 判断追高、破位、回收 |
| 成交额、换手率、量比 | 判断量价确认 |
| MA5、MA10、MA20 | 判断趋势层级 |
| 近20日高低点 | 判断突破和支撑 |

## 输出引用

最终回答必须写明数据来源和获取时间。若使用网页或公开资料，应提供链接。若只能从截图读取，应明确“来源：用户截图”。
