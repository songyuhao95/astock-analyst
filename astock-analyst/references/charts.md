# 图表辅助

只有当用户明确要求生成图表或可视化时才读取本文件。普通持仓建议不需要输出图表。

## 推荐图表

| 场景 | 图表 |
|---|---|
| 持仓复盘 | 仓位分布饼图 |
| 盈亏分析 | 持仓盈亏柱状图 |
| 趋势判断 | 价格 + MA5/MA10/MA20 |
| 风险线说明 | 当前价、支撑、压力、止损线 |

## 最小绘图要求

- 标题包含日期、总资产或标的名称。
- 颜色保持A股习惯：红涨绿跌。
- 坐标、图例、单位必须清楚。
- 图表只服务结论，不替代文字建议。

## 数据字段

持仓图表至少需要：

| 字段 | 说明 |
|---|---|
| 股票名称 | 完整名称 |
| 持有市值 | 用于仓位分布 |
| 持有盈亏 | 用于盈亏柱状图 |
| 仓位% | 用于组合风险 |

趋势图表至少需要：

| 字段 | 说明 |
|---|---|
| 日期 | 日线时间序列 |
| 收盘价 | 趋势主体 |
| MA5/MA10/MA20 | 趋势层级 |
| 成交额或成交量 | 量能确认 |
